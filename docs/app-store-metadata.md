# App Store / Google Play 申請用メタデータ

PWAのまま運用も可能ですが、ストアにあった方が**信頼性UP・発見性UP**します。
最も簡単な方法は「**PWABuilder**」（Microsoft製、無料）でPWA→ネイティブアプリ化。

---

## 推奨アプローチ：PWABuilder

1. https://www.pwabuilder.com にアクセス
2. URL に `https://yorimichi.in-dx.jp/` を入力
3. PWAスコア確認 → 改善提案を実装
4. iOS / Android のパッケージを生成（無料）
5. ストアに申請

これでPWAをそのままアプリ化できます。

### スクリーンショット要件

| サイズ | 用途 | 内容案 |
|---|---|---|
| iPhone 6.7" (1290×2796) | App Store 必須 | ガチャ画面・地図画面・コース詳細・完走証明書・図鑑（5枚） |
| iPad 12.9" (2048×2732) | iPadサポート時 | 同上の横長版 |
| Android Phone | Play Store 必須 | 同上 |
| Android Tablet | 推奨 | 同上 |

スクリーンショット撮影手順：
1. Chrome DevTools → Mobile View（iPhone 14 Pro Max 等）
2. 各画面をスクショ
3. PWABuilder または手動で適切なサイズへ加工

### アイコン要件

| サイズ | 用途 |
|---|---|
| 1024×1024 PNG | App Store / Play Store メイン |
| 各種 | iOS/Android 各端末向け |

現在の絵文字アイコン（👣 + オレンジ円）を SVG → PNG 化したものを使う。
PWABuilder が自動生成してくれます。

## カテゴリ・年齢制限

- **App Store カテゴリ**：旅行（メイン）/ ナビゲーション（サブ）
- **Google Play カテゴリ**：旅行＆地域 / マップ＆ナビ
- **年齢制限**：4+ / 全年齢
- **コンテンツレーティング**：なし
