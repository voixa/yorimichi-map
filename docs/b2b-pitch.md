## 📋 営業の進め方メモ

### 優先順位
1. **個店（直接訪問）** — 当日決まる可能性あり、来店動機にもなる
2. **商店街振興組合** — 紹介の連鎖が起きやすい
3. **メディア** — 1社採用されるとB2B営業が一気に楽になる
4. **自治体** — 時間かかるが大型案件

### スクリプト
- 電話アポ → 15分会いましょう（その日のうち〜翌日）
- 訪問 → スマホでアプリ実演 → ガチャ引いてもらう → 「面白い！」
- 課題ヒアリング → 集客困ってる？回遊促進したい？
- 提案 → 無料掲載でまず実験、効果出たら有料化相談

### 反論切り返し
- 「アプリってどうせ流行らない」
  → 「そうですよね。なので**完全無料**でまず1ヶ月だけ試させてください。流行らなければ撤退します」
- 「うちは古いから関係ない」
  → 「むしろ古い名店こそ、若い世代に**発見**されるべきです。雷おこしの常連は減ってますよね？」
- 「予算がない」
  → 「最初は無料です。**流入が増えてから**手数料モデルにご相談で大丈夫です」

### KPI
- 1週間：アポ取得数（目標 3件）
- 1ヶ月：個店掲載数（目標 5店舗）
- 3ヶ月：有料契約 1件 + 公式採用 1件
