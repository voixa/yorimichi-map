## サービス概要

「寄り道マップ」は、**散歩コースをガチャで引く**というエンタメ要素と、**実際に街を歩いてスタンプを集める**スタンプラリー要素を組み合わせた地図サービスです。

### 主な特徴

1. **ガチャ式コース発見**
   - 大きなガチャマシンUIでハンドルを回すと、その日の散歩コースが1本ポロッと出てくる
   - レアリティはN/R/SR/LRの4段階。早朝5:30限定の「霧の井の頭・早朝禁忌の弁天さま」など隠しコースも

2. **GPS自動チェックイン**
   - スポットに50m以内に近づくと自動でスタンプGET
   - 完走すると「完走証明書」が発行され、X/LINEで自慢可能

3. **ポケモン図鑑型コレクション**
   - 引いていないコースは「？？？」のシルエット表示
   - 集めたくなるコレクション体験

4. **PWA対応**
   - スマホで「ホーム画面に追加」すれば専用アプリのように使用可能
   - オフラインでも基本機能は動作

5. **完全無料・登録不要**
   - 個人情報の入力なし
   - 位置情報は端末内でのみ処理（運営サーバーへの送信なし）

## 開発経緯

運営の柳下征二郎は明治大学2年生在学中で、別事業として飲食店向けAI電話サービス「VOIXA」を経営しています。

「Googleマップは便利すぎて偶然の発見がなくなった。Pokemon GOのように『街を歩くこと自体が遊び』になるサービスを作りたかった」とコメント。

開発には Anthropic 社の AI コーディング環境「Claude Code」を活用。約25時間（3日間）で MVP から本番公開まで完了させた、**個人での爆速サービス立ち上げ**の最新事例となっています。

技術スタック：
- フロントエンド：Vanilla JavaScript + Leaflet.js
- 地図データ：OpenStreetMap
- ホスティング：Google Cloud Run
- 経路計算：OSRM
