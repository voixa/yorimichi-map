            ❓ ヘルプ・よくある質問

            街歩きガチャって何？

            街の散歩コースを ガチャ形式で1本だけ引いて、GPSスタンプラリーで遊べる地図サービスです。あなたの街にあるまだ知らないスポットを発見できます。

            無料で遊べる？

            はい、毎日3回まで無料でガチャを引けます。それ以降は1ガチャ=3コイン（コインショップで購入）。月額¥480の プレミアムプラン なら無制限に引けます。

            位置情報は安全？

            位置情報は あなたの端末内でのみ 処理されます。運営サーバーには送信していません。GPSはスタンプラリーのチェックインだけに使います。

            「自由ルート」と「コース」の違いは？

            📖 既存コース：吉祥寺・谷根千・神保町・浅草のキュレーション済みコース（13本）
            🎯 自由ルート：出発地と目的地を入れるとAIがランダムにコース生成
            🌿 散歩：出発地のみ入れて気の向くままに

            コインの使い道は？

            無料ガチャ3回を使い切った後、追加でガチャを引くのに必要です（1ガチャ=3コイン）。コインは 毎日のログインボーナス・コース完走・友達招待 でも貯まります。

            友達と一緒に遊びたい

            プロフィール画面の「🎁 友達を招待」リンクをコピーして送ってください。招待された人が初ガチャを引くと あなた・友達ともに +15🪙。

            歩きスマホは大丈夫？

            歩きスマホは危険です。本サービスは 音声ガイド と ホーム画面ロック中も動作 する設計です。HUDの 🔊 ボタンで音声案内をオン推奨。

            データを別端末に移したい

            プロフィール → ⚙️データ管理 → 📤 エクスポート でJSON保存 → 移行先で 📥 インポート。

            サブスクの解約方法は？

            コインショップ画面下部の「プレミアム会員」セクションから「プランを解約」を押してください。期間末まで利用可能です。

            不具合報告・問い合わせ

            📧 info@in-dx.jp までメールください。

            街歩きガチャへようこそ

            あなたの街の散歩コースを ガチャで引いて、GPSスタンプラリーで遊べる地図サービスです。

            「あと◯分あったら、どこ寄る？」

            🎰

            巨大ガチャで1本ゲット

            ハンドルを回すとカプセルが1つコトン。中には レアリティ付きの散歩コース が入ってます。

            N → R → SR → ✨LR の4段階。LRには「早朝5:30限定」のような隠しコースも。

            🗺️

            3つの遊び方

            • 📖 既存コース：吉祥寺・谷根千・神保町・浅草のキュレーション済みコース
            • 🎯 自由ルート：出発地→目的地を入れるとAIがランダムに街歩きコースを生成
            • 🌿 散歩：出発地のみ入れて気の向くまま
            🎁

            初回特典：SR以上確定

            最初の1引きは SR以上が出るまでハズレなし。さらに、毎日ガチャ無料3回 + 連続ログインボーナス！

            「歩き始める」を押すと GPS で自動スタンプ。完走すると証明書ゲット 🏅

            位置情報の利用について

            • ✅ 用途：スポット到着の自動チェックインのみ
            • ✅ 送信先：どこにも送らず、端末内だけで処理
            • ✅ 保存先：このブラウザの localStorage のみ
            • ⚠ 拒否しても手動チェックインボタンで遊べます

            「許可」を押すとブラウザの確認ダイアログが出ます。

            🔒 プライバシーポリシー

            収集する情報

            本サービスはアカウント登録不要で、以下のデータのみをあなたの端末内で扱います：

            • 位置情報（GPS）：自動チェックインのみで利用、送信なし
            • 発見コース・完走コース・連続ログイン日数：localStorage に保存
            • ガチャ回数・コイン残高：localStorage に保存

            第三者送信

            地図表示・経路計算のため、以下に位置座標を送ります：

            • OpenStreetMap（タイル画像）
            • OSRM（ルート計算）
            • Nominatim（住所検索）

            外部リンク

            「Googleマップで開く」「𝕏で共有」「LINEで送る」を押すと外部サイトが開きます。各サービスのポリシーに従います。

            クッキー

            使用していません。localStorage と Service Worker キャッシュのみ。

            データ削除

            ブラウザの「サイトデータをクリア」で全削除できます。
